About me

An aspiring software developer based in Yorkshire

I was introduced to coding during University, whilst studying Environmental Geoscience. This started as using MATLAB and eventually Python for data analysis and building simple models. I was instantly drawn to the practice and completed my final dissertations using both MATLAB and python to model flood risk and analyse seismic waveforms.

Since leaving University I continued developing my skills, undertaking online courses to sharpen my Python abilities, as well as in C++ to get experience in another language.

I've worked as a Remediation Engineer and Project Manager for over 5 years now, and have managed to use some of these skills to improve my workflow - from building simple models to calculate flow characteristics to web scraping to extract monitoring data from cloud based services.

Outside of working life I'm a keen cook, and if not creating in the kitchen I'm currently creating and recording new music for our bands second album
